## How to Run

1. Open `FinalProject/` and launch the Google Colab notebook
2. Click **Runtime → Run All** at the top or run the cells sequentially.
3. The notebook will automatically download the MNIST dataset, train the model, and evaluate it against both the MNIST test set and the class handwritten digit dataset
